## Resumo
Este quarto trabalho consiste no desenvolvimento de uma ontologia OWL para a biblioteca fictícia "Entre Ontem e Amanhã", focada na organização de livros em múltiplas linhas temporais. O objetivo é modelar conceitos como livros, autores e eventos, utilizando propriedades e restrições para permitir a inferência automática e a classificação de indivíduos pelo reasoner. O projeto inclui ainda a resolução de algumas questões através de queries SPARQL 1.1 para extrair dados complexos e identificar inconsistências semânticas na base de conhecimento.
